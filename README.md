🎬 Netflix India Clone
A pixel-perfect clone of the Netflix India landing page. This project focuses on modern CSS techniques, responsive design, and UI/UX replication.

✨ Features
Responsive Design: Optimized for all screen sizes (Desktop, Tablet, Mobile).

Separation Curve: Custom CSS radial-gradient curve for a premium look.

Trending Slider: Horizontal movie slider with stylized background numbering.

Modern UI: Interactive FAQ accordion and glassmorphic card layouts.

Custom Fonts: Integrated 'Geologica' font for authentic branding.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Flexbox, Grid, Media Queries, and Custom Animations.

Assets: SVG icons and high-quality movie posters stored in the Assest/ folder.
